About the Dark Web

The dark web emerged in the early 2000s as part of research on anonymous communication. It has been used by governments, activists, and citizens to communicate securely without fear of surveillance. Today, it is used both for legitimate purposes (privacy, free speech in restricted countries) and illegal activities like drug, weapon, and human trafficking. Despite its reputation, using the dark web is not illegal by itself. Access requires a special browser called Tor, which uses "onion routing" to hide IP addresses and keep communication private.

Unit 1: Data

bit: A binary digit, either 0 or 1.

byte: A sequence of 8 bits.

roundoff: Error from not enough bits to represent numbers fully.

analog data: Values that change smoothly over time.

lossless: Compression that preserves all data.

lossy: Compression that discards some data.

metadata: Data about data, e.g., file information.

Unit 2: Computer Systems and Networks

Computing Device: A device that runs programs (computer, phone).

Computer Network: A group of connected computing devices.

Path: The connections from sender to receiver.

Protocol: Agreed rules for communication.

Fault Tolerant: Keeps working even if parts fail.

Scalability: Ability to grow/shrink to meet demand.

Bandwidth: Maximum transmission capacity (bit rate).

URL: A human-readable web address.

IP: Assigns device addresses and routes data.

TCP/UDP: Protocols for reliable/unreliable delivery.

Packets: Chunks of data messages split into for delivery.

Routers: Devices that forward packets.

DNS: Translates domains into IP addresses.

HTTP: Protocol for requesting web pages.

WWW: System of linked pages on HTTP.

Redundancy & Fault Tolerance: Internet keeps working even if parts fail.

IPv4 vs IPv6: IPv6 provides massively more addresses.
